

# SAKN's Edits — Coming Soon Splash Screen

## Overview
A sleek, dark-themed "coming soon" splash page inspired by kenos.lol, featuring the brand name **sakn's edits** with smooth animations.

## Design
- **Dark background** with a minimal, centered layout
- **Terminal/code icon** at the top
- **"sakn's edits"** as the main brand text, bold and prominent
- **"Coming Soon"** or **"Upcoming"** subtitle text with a **wavy animation effect** — the text gently waves/ripples continuously
- Clean, modern typography with subtle glow or gradient accents

## Animations
- **Fade-in** entrance for the icon and brand name
- **Wavy/ripple animation** on the "coming soon" text — letters undulate smoothly in a wave pattern
- Subtle pulsing or glow effect on the icon for added polish

## Page Behavior
- This is the entire landing page (no navigation away)
- Fully responsive for mobile and desktop
